Transmitir Caractere ID do Código#

Um caractere de ID de código identifica o tipo de código de um código de barra escaneado. Isso pode ser útil quando decodifica mais de um tipo de código. O caractere ID do código é inserido entre o caractere de prefixo (se selecionado) e o símbolo decodificado.

Identificadores do código ID#

Identificadores de código AIM#

Cada identificador de código AIM contém a string de três caracteres ]cm onde:

  • ] = Caractere Bandeira

  • c =Caractere de Código (ver tabela)

  • m= Caractere Modificador
